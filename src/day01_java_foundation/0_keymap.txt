    运行--Alt + f9
    选择运行--Alt + shift + f9

    调试--Alt + f10
    选择调试--Alt + shift + f10

    设置/取消当前行断点--Alt + d (断点必须设置在有初始化的代码行)
    调试进入执行语句--f7
    调试逐条执行语句--f8
    调试至下个断点处--f9

    光标左移--alt+j
    光标右移--alt+l
    光标下移--alt+k
    光标上移--alt+i

    光标返回--alt + b
    光标前进--alt + n （next代替forward，因为Alt+F用于在所有文件中查找内容）

    向左滚动--alt + shift + j
    向右滚动--alt + shift + l
    向上滚动--alt + shift + i
    向下滚动--alt + shift + k

    左右切换编辑文件--Alt+左右箭头

    向上下左右选中--shift+上下左右箭头

    terminal-- Alt+f12
    project -- Alt + 1
    run 窗口-- Alt + 4  # 需要输入数据时，直接进入命令行输入。
    debug 窗口-- Alt + 5 # 只能显示和隐藏debug大窗口，不能进入其中的console小窗口输入数据，要进入console小窗口输入数据
                           需要再按一次tab键。
    注释当前行--Ctrl + /

    选择选项卡 Ctrl + tab  alt + 左右箭头
    关闭选项卡 Alt + g (close tab)--editor tab
    关闭活动选项卡 shift + g (close active tab)--tools window tab（比如底部的命令行窗口）

    git操作：
        1.进入提交commit界面--shift+C
        2.提交并推送commit and push--Alt+shift+C
        3.确认推送push--Alt+P

        git文件名的颜色
            未加入git---红色
            加入到git---绿色
            提交到本地---蓝色
            推送到远程---无色

    删除当前行：Ctrl + Y
    复制当前行：Ctrl + C
    重复当前行：Ctrl + D

    另起一行：shift + enter

    在当前文件中查找 -- Ctrl + F
    在当前文件中替换 -- Ctrl + R
    在所有文件中查找 -- Alt + F
    查找项目文件 -- 双击shift

    选中部分代码，让其被if/try/while等结构包围 -- Ctrl + Alt + S
    选中部分代码，让其上下移动 -- Ctrl + Shift + 上/下箭头
